## 3. 列表

### 3.1 无序列表

- 语法

```
* 无序列表1
+ 无序列表2
- 无序列表3
```

- 效果

- 无序列表1

- 无序列表2

- 无序列表n

### 3.2 多行无序列表

- 语法

```
* 多行无序序列1
TAB * 多行无序序列2
TAB TAB * 多行无序序列3
```
- 效果

- 多行无序序列1
- 多行无序序列2
- 多行无序序列3

### 3.3 有序列表

- 语法

```
1. 有序序列1
2. 有序序列2
3. 有序序列3
```

- 效果

2. 有序列表2
3. 有序列表3
